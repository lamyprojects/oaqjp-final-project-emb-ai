# Final project

## AI-Based Web Application Development and Deployment

This project implements an AI-based Emotion Detection web application using the embedded Watson NLP library.

The application performs the following tasks:

- Emotion detection using Watson NLP
- Output formatting and dominant emotion extraction
- Application packaging as a Python module
- Unit testing validation
- Web deployment using Flask on localhost:5000
- Error handling for blank input entries
- Static code analysis using PyLint with 10/10 compliance

The system processes customer feedback in text format and returns emotion scores for:

- Anger
- Disgust
- Fear
- Joy
- Sadness

Along with the dominant emotion detected.

This repository contains all required tasks (Task 1 – Task 8) as specified in the Final Project guidelines.
